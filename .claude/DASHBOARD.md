## 调用优先级

当需要某个功能时，**优先使用 Skill**，降级方案：

| 功能 | 优先 | 降级 |
|------|------|------|
| 搜索 | skills → baidu-search/web-reader | MCP 全局搜索 |
| 文生图 | skills → qwen-Image | MCP 全局 |
| 文档转换 | skills → markdown-converter | MCP 全局 |
| 网页 | skills → web-reader | MCP 全局 |

> 技能文件位置：`.claude/skills/`（本地） + `~/.claude/skills/`（全局）
